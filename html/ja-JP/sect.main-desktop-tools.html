13.4. 電子メール

13.4.1. Evolution

COMMUNITY 人気のあるパッケージ

popularity-contest パッケージをインストールすると、最も人気のあるパッケージを Debian プロジェクトに通知する目的の自動調査に参加することになります。スクリプトは cron によって毎週実行され、結果を (HTTP または電子メールで) 送信します。結果には匿名化されたインストール済みパッケージのリストとパッケージに含まれるファイルへの最新のアクセス日が含まれます。このおかげで、インストール済みパッケージから実際に使われているパッケージを識別することが可能です。
Debian プロジェクトはパッケージの人気度を参考にして作業を行います。パッケージの人気度は 1 枚目のインストールディスクに含めるパッケージを決定するために使われます。パッケージの人気度は、ユーザ数の極端に少ないパッケージをディストリビューションから削除するか否かを決定する際にも、重要な要素になります。popularity-contest パッケージをインストールして、調査に参加することを心から推奨します。
収集されたデータは毎日公開されています。
→ http://popcon.debian.org/
これらの統計は同等と思われる 2 種類のパッケージを選ぶ際の手助けになります。より人気のあるパッケージを選べば、それが正しい選択となる可能性を高めることになります。
Evolution は GNOME 電子メールクライアントで、apt-get install evolution を使ってインストールできます。Evolution は単純な電子メールクライアントに留まらず、カレンダー、アドレス帳、タスクリスト、メモ (自由形式ノート) アプリケーションの機能も持っています。電子メール部分は強力なメッセージ索引システムを備え、すべてのアーカイブ済みメッセージを対象とする検索クエリに基づいて仮想フォルダを作成することが可能です。言い換えると、すべてのメッセージは同じ方法で保存されますが、フォルダ的なものの中に表示されます。各フォルダにはフィルタリング条件群に一致するメッセージが含まれます。

図13.4 Evolution 電子メールソフトウェア

Evolution の拡張機能を使うことで、Microsoft Exchange 電子メールシステムと統合することが可能です; これを行うには、evolution-exchange パッケージが必要です。

13.4.2. KMail

KDE 電子メールソフトウェアは apt-get install kmail を使ってインストールすることが可能です。KMail は電子メールだけを取り扱いますが、KDE-PIM (Personal Information Manager) と呼ばれるソフトウェアスイートに所属します。KDE-PIM には、アドレス帳、カレンダーなどの機能も含まれます。KMail は強力な電子メールクライアントに期待されるすべての機能を持っています。

図13.5 KMail 電子メールソフトウェア

13.4.3. Thunderbird と Icedove

この電子メールソフトウェアは icedove パッケージに含まれ、Mozilla ソフトウェアスイートの一部です。さまざまな言語に翻訳されており、これらは icedove-l10n-* パッケージから利用できます。enigmail 拡張を導入すれば、メッセージを暗号化したり署名することが可能です (悲しいかな、これはすべての言語向けに利用可能ではありません)。

図13.6 Icedove 電子メールソフトウェア

Thunderbird は最良の電子メールクライアントの 1 つで、Mozilla Firefox と同じく偉大なる成功を収めています。
厳密に言えば、Debian Wheezy には Icedove が含まれますが、Thunderbird は含まれません。これには法律上の理由があり、詳細は傍注CULTURE Iceweasel、Firefox、その他で説明します; しかし名前 (とアイコン) を除けば、Iceweasel と Firefox は同じものです。
