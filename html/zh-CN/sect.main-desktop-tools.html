13.4. 邮件

13.4.1. Evolution

社区 流行 软件包

安装 popularity-contest 软件包会参与最流行软件包的自动调查。 cron 会每周运行一个脚本，并匿名发送（通过 HTTP 或电子邮件）安装的软件包列表，以及最后读取时间。这样可以区分已安装的软件包中，哪些是实际使用的。
This information is a great help to the Debian project. It is used to determine which packages should go on the first installation disks. The installation data is also an important factor used to decide whether to remove a package with very few users from the distribution. We heartily recommend installing the popularity-contest package, and participating in the survey.
所收集的数据会每日公布。
→ http://popcon.debian.org/
这些统计数据会便于在两个看起来等效的软件包之间做出选择。选择的软件包越流行，做出的选择可能越好。
Evolution 是 GNOME 中的电子邮件客户端，可以使用 apt-get install evolution 安装。 Evolution 不仅仅是一个邮件客户端，也提供日历，地址本，任务列表，和记事本应用程序。它的电子邮件组件包含了强大的信息索引系统，并允许基于搜索在所有存档信息中创建虚拟文件夹。换句话说，所有信息都以相同的方式储存，但以文件夹的组织方式显示，每个文件夹中包含了符合过滤条件的信息。　

图 13.4. Evolution 电子邮件软件

Evolution 扩展插件可以集成到 Microsoft Exchange 邮件系统；需要安装 evolution-exchange软件包。

13.4.2. KMail

KDE 邮件软件可以通过 apt-get install kmail安装。KMail 只处理电子邮件，属于 KDE-PIM （ Personal Information Manager）软件包，该软件包包含了地址本，日历组件，等待。KMail 具备了所有电子邮件客户端的优良特性。

图 13.5. KMail 电子邮件软件

13.4.3. Thunderbird 和 Icedove

这个邮件软件，包含在 icedove 软件包中，是 Mozilla 软件套装的一部分。各种本地化套装在 icedove-l10n-* 软件包中； enigmail 插件处理信息加密和签名（并不适用于所有语言）。

图 13.6. Icedove 电子邮件软件

Thunderbird 是最好的邮件客户端之一，并获得极大成功，就如同火狐浏览器。
Strictly speaking, Debian Wheezy contains Icedove, and not Thunderbird, for legal reasons we will detail in sidebar 文化 冰熊，火狐及其他 later on; but apart from their names (and icons), there are no real differences between them.
